# Retail Data Analysis and Visualization

## Overview
This project performs an analysis of an online retail dataset using PySpark and Python's visualization libraries (Matplotlib and Seaborn). The objective is to extract meaningful insights regarding revenue generation, monthly sales trends, and order quantity distributions.


### Key Analysis Metrics:
1. **Total Revenue by Country:** Aggregates and finds the top 10 countries generating the most revenue. 
2. **Monthly Revenue Trend:** Tracks the revenue generated over time on a monthly basis.
3. **Distribution of Order Quantities:** Analyzes the distribution of ordered quantities.
